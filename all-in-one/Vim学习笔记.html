Vim学习笔记

一、vim基本操作

1.1 打开文件

打开文件可以在terminal中输入：

vim [filename]

此时如果文件不存在则会被创建，打开文件后进入的是编辑模式，可以执行一些vim的命令。 1.2 跳转指令

光标的跳转：

xxxxxxxxxx
k                上移一个字符
j                下移一个字符
h                左移一个字符
l                右移一个字符
ctrl+f           在文件中前移一页(相当于 page down)
ctrl+b           在文件中后移一页(相当于 page up)

在当前行内移动：

xxxxxxxxxx
w                右移光标到下一个字的开头
e                右移光标到一个字的末尾
b                左移光标到前一个字的开头
0                数字０，左移光标到本行的开始
$                右移光标，到本行的末尾
^                移动光标，到本行的第一个非空字符

在当前的屏幕范围内跳转：

xxxxxxxxxx
H                将光标移到屏幕上的起始行（或最上行）
2H               将光标移到屏幕的第２行
M                将光标移到屏幕中间
L                将光标移到屏幕最后一行
3L               将光标移到屏幕的倒数第3行

更大范围的跳转：

xxxxxxxxxx
*                在当前文件内搜索光标所指字符，并跳转到下一处
#                在当前文件内搜索光标所指字符，并跳转到上一处
(/)              移动到 前/后 句 的开始
{/}              跳转到 当前/下一个 段落 的开始
g_               到本行最后非空字符处
fa               到下一个为 a 的字符处，a可以改成其他字符
t,               到逗号前的第一个字符。逗号可以变成其它字符
3fa              在当前行查找第三个出现的 a
F/T              和 f 和 t 一样，只不过是相反方向
gg               将光标定位到文件第一行起始位置
G                将光标定位到文件最后一行起始位置
NG或Ngg          将光标定位到第 N 行的起始位置

 

除了常规的跳转，还可以使用搜索来跳转到所需的位置，vim中的搜索是键入字符/，后面跟以要搜索的字符串，然后按回车键，然后使用n来寻找下一个匹配的字符串：

xxxxxxxxxx
/str1            正向搜索字符串 str1
n                继续搜索，找出 str1 字符串下次出现的位置
N                继续搜索，找出 str1 字符串上一次出现的位置
?str2            反向搜索字符串 str2

vim还支持用正则表达式来搜索，例如要搜索以apt开头的字符串：

xxxxxxxxxx
:/^apt/

上面的^表示匹配字符串的开始，一下给出一些常用的匹配命令：

xxxxxxxxxx
^                放在字符串前面，匹配行首的字
$                放在字符串后面，匹配行尾的字
\<               匹配一个字的字头
\>               匹配一个字的字尾
.                匹配任何单个正文字符
[str]            匹配 str 中的任何单个字符
[^str]           匹配任何不在 str 中的单个字符
[a-b]            匹配 a 到 b 之间的任一字符
*                匹配前一个字符的 0 次或多次出现
\                转义后面的字符

想了解更多的正则匹配命令，参阅：

《正则表达式30分钟入门》

1.3 插入模式

用vim打开文件后只能执行1.2所述命令，而不能编辑，要编辑文件，必须先选一种插入模式，vim有以下几种编辑模式：

xxxxxxxxxx
i                在光标左侧插入正文
a                在光标右侧插入正文
o                在光标所在行的下一行增添新行
O                在光标所在行的上一行增添新行
I                在光标所在行的开头插入
A                在光标所在行的末尾插入

如果在插入模式下想要进行查找、复制等操作，要先退出插入模式，退出插入模式的方法是，按ESC键或组合键Ctrl+[。

1.4 文本操作

复制命令是yy

xxxxxxxxxx
yy               复制当前行到内存缓冲区
nyy              复制 n 行内容到内存缓冲区
5yy              复制 5 行内容到内存缓冲区
“+y              复制 1 行到操作系统的粘贴板
“+nyy            复制 n 行到操作系统的粘贴板

 

粘贴命令是p

xxxxxxxxxx
p                小写字母 p，将缓冲区的内容粘贴到光标的后面
P                大写字母 P，将缓冲区的内容粘贴到光标的前面

 

删除命令是 dd(删除行)、 x(删除字符)

xxxxxxxxxx
x                删除光标所指向的当前字符
dd               删除光标所在行
nx               删除光标所指向的前 n 个字符
dw               删除光标右侧的单词
ndw              删除光标右侧的 n 个单词
db               删除光标左侧的单词
ndb              删除光标左侧的 n 个单词
ndd              删除 n 行内容，并去除空隙
d$               从当前光标起删除字符直到行的结束
d0               从当前光标起删除字符直到行的开始
J                删除本行的回车符（CR），并和下一行合并

要注意的是，以上所说的删除都是指剪切，也就说如果缓存内有内容，则使用上述命令会覆盖缓存区原来的内容

 

撤销和重复

xxxxxxxxxx
u                撤消
.                重复

 

替换命令有r、c和s，替换命令执行以后，通常会由编辑模式进入插入模式

xxxxxxxxxx
rc               用 c 替换光标所指向的当前字符
nrc              用 c 替换光标所指向的前 n 个字符
s                用输入的正文替换光标所指向的字符
S                删除当前行，并进入编辑模式
ns               用输入的正文替换光标右侧 n 个字符
nS               删除当前行在内的 n 行，并进入编辑模式
cw               用输入的正文替换光标右侧的字
cW               用输入的正文替换从光标到行尾的所有字符（同 c$ )
ncw              用输入的正文替换光标右侧的 n 个字
cb               用输入的正文替换光标左侧的字
ncb              用输入的正文替换光标左侧的 n 个字
cd               用输入的正文替换光标的所在行
ncd              用输入的正文替换光标下面的 n 行
c$               用输入的正文替换从光标开始到本行末尾的所有字符
c0               用输入的正文替换从本行开头到光标的所有字符

二、命令模式

2.1 创建、保存与退出

在编辑模式下键入:，光标就跳到屏幕最后一行，并在那里显示冒号，此时已进入命令模式。

在已经启动的vim中打开一个文件可以使用以下命令：

xxxxxxxxxx
:e directory/filename

 

保存编辑完的文件：

xxxxxxxxxx
:file filename    指定保存的文件名(新建时没有指定文件名才需要)
:w                保存文件
:w filename       将文件另存为指定的名字
:wq               保存并退出(相当于ZZ)

 

退出：

xxxxxxxxxx
:q                在未作修改的情况下退出；
:q!               放弃所有修改，退出编辑程序。

 

重命名：

xxxxxxxxxx
:f file           将当前文件重命名为 file
:f                打印当前文件名称和状态，如文件的行数、光标所在的行号等

2.2 跳转

在命令模式下也能完成编辑模式下ngg(跳转到指定行)的操作

xxxxxxxxxx
:n               将光标移到第 n 行

命令模式下，可以规定命令操作的行号范围，除指定数字行号外，还可以用.表示当前行，$表示正文最后一行的行号，.+3表示当前行再往下3行，这些写法在搜索、插入等操作时指定行号范围很有用。

2.3 插入文件

在命令模式下，允许从文件中读取正文，或将正文写入文件

xxxxxxxxxx
:aw file         将第 a 行写入 file 文件
:a,bw file       将第 a 行至第 b 行写入 file 文件(a、b可以是"."、"$"等)
:r file          读取 file 文件的内容，插入当前光标所在行的后面

2.4 搜索和替换

和编辑模式一样，命令模式也可以搜索，而且可以结合其他指令一起使用：

xxxxxxxxxx
:/str/                  正向搜索，将光标移到下一个包含字符串 str 的行
:?str?                  反向搜索，将光标移到上一个包含字符串 str 的行
:/str/w file            正向搜索，并将第一个包含字符串 str 的行写入 file 文件
:/str1/,/str2/w file    正向搜索，并将包含字符串 str1 的行至包含字符串 str2 的行写 file 文件

替换命令是:s：

xxxxxxxxxx
:%s/str1/str2/        用字符串 str2 替换行中首次出现的字符串 str1
:s/str1/str2/g        用字符串 str2 替换行中所有出现的字符串 str1
:.,$ s/str1/str2/g    用字符串 str2 替换正文当前行到末尾所有出现的字符串 str1
:1,$ s/str1/str2/g    用字符串 str2 替换正文中所有出现的字符串 str1
:g/str1/s//str2/g     用字符串 str2 替换正文中所有出现的字符串 str1
:m,ns/str1/str2/g     将从m行到n行的str1替换成str2

其中%表示替换所有行，s表示替换命令，g放在命令末尾，表示替换当前行中所有匹配的字符串；不加g，表示替换首次匹配的字符串；g放在命令开头，表示在全文范围内中替换匹配的符串

统计指定字符串在全文出现的次数：

xxxxxxxxxx
:%s/str/&/gn

类似地，可以使用搜索指令+其他操作指令(如删除、插入等)来进行操作

三、vim的配置与插件

3.1 选项配置

vim的设置指令是:set，常用的功能有：

xxxxxxxxxx
:set autoindent        设置该选项，则正文自动缩进
:set ignorecase        设置该选项，则忽略规则表达式中大小写字母的区别
:set number            设置该选项，则显示正文行号
:set ruler             设置该选项，则在屏幕底部显示光标所在行、列的位置
:set tabstop           设置按 Tab 键跳过的空格数。例如 :set tabstop=n，n 默认值为 8
:set mk                将选项保存在当前目录的 .exrc 文件中

 

在vim中可以直接执行shell指令，这样在编写代码时就不用先退出再编译代码了：

xxxxxxxxxx
:!command              执行command

3.2 分屏与分标签

在已经启动的vim中使用分屏命令：

xxxxxxxxxx
:split（可用缩写 :sp）            上下分屏
:vsplit（可用缩写 :vsp）          左右分屏

在启动vim时带分屏参数：

xxxxxxxxxx
vim -On file1 file2...   打开 file1 和 file2 ，垂直分屏
vim -on file1 file2...   打开 file1 和 file2 ，水平分屏

切换分屏：

xxxxxxxxxx
Ctrl+w+h            切换到当前分屏的左边一屏
Ctrl+w+l            切换到当前分屏的右边一屏
Ctrl+w+j            切换到当前分屏的下方一屏
Ctrl+w+k            切换到当前分屏的上方一屏
Ctrl+w+w            按照逆时针方向跳转分屏

新建标签页，即标签页操作：

xxxxxxxxxx
:tabnew [filename]      新建标签页(并在标签页内加载指定文件)
:tN[ext]                跳转到上一个匹配的标签
:tabN[ext]              跳到上一个标签页
:tabc[lose]             关闭当前标签页
:tabdo                  为每个标签页执行命令
:tabe[dit]              在新标签页里编辑文件
:tabf[ind]              寻找 'path' 里的文件，在新标签页里编辑之
:tabfir[st]             转到第一个标签页
:tabl[ast]              转到最后一个标签页
:tabm[ove]  N           把标签页移到序号为N位置
:tabnew [filename]      在新标签页里编辑文件
:tabn[ext]              转到下一个标签页
:tabo[nly]              关闭所有除了当前标签页以外的所有标签页
:tabp[revious]          转到前一个标签页
:tabr[ewind]            转到第一个标签页

3.3 视图模式

在vim中有三种视图模式：

xxxxxxxxxx
v                       字符选择模式
V                       行选择模式
Ctrl+v                  块选择模式

对于块选择模式的操作方法：

xxxxxxxxxx
第一步：将光标停在块开始的位置
第二步：ctrl + v 进入块选择模式
第三步：Ngg 让光标停在指定行，这样就选中了若干列
第四步：按方向键的 左/右 箭头， 增加/减少 选中的列，这样就选中了多列
第五步：按 d 或者 x 删除选中的内容，按 r 将选中块中的字符全部替代为某个字符，按 > 将块向右移动一个制表符

3.4 diff和二进制文件查看

类似linux下的diff，vim也有diff指令：

vimdiff file1 file2     在terminal下使用vim的diff功能
:diffsplit file2        在vim中将file2与当前打开文件进行对比
:vert diffsplit file2   上下分屏模式的diff
:vert diffpatch file2   上下分屏模式的patch
:only                   返回原来编辑的文件
:diffu[pdate]           更新

打开二进制文件：

xxxxxxxxxx
vim -b filename         在终端里以二进制模式打开一个文件
:%!xxd                  以二进制+原文本的形式显示二进制文件内容
:%!xxd -r               返回普通模式
:%!xxd -g 2             每2个字节为1组(默认)
